# Music Vinyl Shop

A Flutter e-commerce app for browsing and purchasing music vinyl records.

## Features

- **ListWheelScrollView**: Browse vinyl records with a wheel scroll effect
- **DraggableScrollableSheet**: Draggable bottom sheet with featured vinyls
- **CarouselScrollSheet**: Carousel slider showcasing featured albums
- **Contact Cards**: Scrollable list of cards with name, email, and phone number
- **Shopping List**: Shopping list with dividers
- **To-Do List**: Interactive to-do list where items can be added using a TextField and button
- **Widget List**: Display of various Flutter widgets in a scrollable column
- **Product List**: Custom scrollable product list with prices and "Buy" buttons

## Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK

### Installation

1. Navigate to the project directory
2. Install dependencies:
   ```
   flutter pub get
   ```

3. Run the app:
   ```
   flutter run
   ```

## Dependencies
